# RahulSpace - Personal Portfolio Website

A modern, responsive portfolio website with futuristic design and playful animations.

## 🚀 Features

- **Futuristic Design**: Modern glassmorphism UI with glowing effects
- **Dark/Light Mode**: User-controlled theme toggle with localStorage persistence
- **Floating Cat Animation**: Interactive cursor-following cat with toggle control
- **Fully Responsive**: Optimized for all device sizes
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Modular Architecture**: Organized file structure for easy maintenance
- **Fast Loading**: Optimized performance with lazy loading
- **SEO Friendly**: Semantic HTML and meta tags
